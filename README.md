# Media Authoring with Java API (MAJ)

The MAJ API is a pure Java API for reading and writing MXF (SMPTE 0377), [AAF](http://www.amwa.tv/projects/MS-01.shtml) 
and Reg-XML (SMPTE 2001). MXF files are commonly used are a container for professional media file formats and AAF is 
supported by a number of professional video editing packages. MXF and Reg-XML are also part of the Interoperable Mastering
Format suite of specifications.
